# My Expenses (Мои расходы)

My Expenses — настольное приложение, разработанное с использованием Python, Tkinter и SQLite, предназначенное для ведения учёта личных расходов. Программа позволяет сохранять информацию о тратах, распределять их по категориям и просматривать общую статистику расходов.

## Функциональные возможности

- Добавление расходов — сохранение информации о совершённых тратах.
- Категории расходов — распределение расходов по категориям («Еда», «Транспорт», «Развлечения»).
- Просмотр списка расходов — отображение всех сохранённых записей.
- Удаление записей — удаление ошибочно введённых расходов.
- Автоматический подсчёт статистики — расчёт общей суммы расходов.
- Хранение данных в SQLite — сохранение информации между запусками программы.
- Простой графический интерфейс — удобная работа без использования командной строки.

## Инструкция по установке

### 1. Скачивание проекта

Скачайте архив с приложением.

### 2. Распаковка файлов

Если был скачан архив, распакуйте его в любую удобную папку.

### 3. Запуск приложения

Перейдите в папку приложения и запустите файл:

```
MyExpenses.exe
```

После запуска откроется главное окно программы.

### 3. Алтернативный запуск

Для альтернативного запуска приложения необходимо наличие:

- Python 3.13 или выше;
- библиотек:
  - sqlite3
  - tkinter
  - datetime

Указанные библиотеки входят в стандартный комплект Python и не требуют дополнительной установки.

После утсановки Python, перейдите в папку проекта и запустите файл:

```
__main_.py
```
Для правильной работы крайне необходима целостность всех файлов приложения.
После запуска откроется главное окно программы.

# Использование программы

## Главное окно

После запуска приложения пользователь увидит главное окно.

В главном окне доступны следующие элементы:

- поле ввода суммы расхода;
- список категорий;
- поле комментария;
- кнопка **Добавить**;
- кнопка **Удалить**;
- список последних расходов;
- кнопка перехода к статистике.

---

## Добавление расхода

Для добавления новой записи необходимо:

1. Ввести сумму расхода.
2. Выбрать категорию.
3. При необходимости добавить комментарий.
4. Нажать кнопку **Добавить**.

После выполнения операции запись будет сохранена в базе данных и появится в списке расходов.

---

## Удаление расхода

Для удаления записи необходимо:

1. Выбрать нужную запись из списка.
2. Нажать кнопку **Удалить**.
3. Подтвердить удаление.

После удаления запись исчезнет из списка и перестанет учитываться в статистике.

---

## Просмотр деталей расходов

Для каждой записи отображаются:

- сумма;
- категория;
- дата произведения расхода;
- Дата записи
- комментарий.

Список автоматически обновляется после каждого изменения данных.

---

## Просмотр статистики

Для просмотра статистики необходимо нажать кнопку **Статистика**.

В окне статистики отображаются:

- общая сумма расходов за день;
- общая сумма расходов за неделю;
- сумма расходов по категориям;
- общая сумма всех расходов.

Статистика пересчитывается автоматически после каждого изменения данных.
